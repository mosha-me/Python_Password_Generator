# 🔐 Python Password Generator

A simple and secure password generator built with Python. 
Generates strong random passwords with letters, digits, and special characters.

## 🚀 Features

- Generates random passwords
- Customizable length
- Includes uppercase, lowercase, digits & special characters
- Security check for weak passwords
- Super simple code (only 15 lines!)

## 📦 Requirements

- Python 3.x
- No external libraries needed!

## 🔧 Installation & Usage

```bash
# Clone the repository
git clone https://github.com/mosha-me/py.git

# Navigate to the folder
cd py

# Run the script
python password_generator.py

```
